Diamond Ring Collector Game!

Here are the Rules


You are given a random number at the start of the game to match. There will be four diamond rings listed below. Each diamond ring is associated with a number. Each time when you click on the diamond ring, the number associated with the ring will add to your score. The object of the game is to get the total score that was given. Each time when the game starts, the game you change the valus of the diamond. If you go over the number you have to match, you lose.
